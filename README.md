# marimo-analysis
# Supply Chain Correlation Analysis


This repository contains:
- `correlation.csv`: Correlation matrix of supply chain metrics
- `heatmap.png`: Excel heatmap with Red-White-Green conditional formatting

Metrics analyzed:
- Supplier_Lead_Time
- Inventory_Levels
- Order_Frequency
- Delivery_Performance
- Cost_Per_Unit

## Tools Used
- Python
- Pandas
- Excel (heatmap visualization)

## Key Outcome
Identified relationships between lead time, inventory, delivery performance, and cost metrics using correlation analysis.
